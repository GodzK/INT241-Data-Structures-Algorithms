# INT241 – Data Structures & Algorithms

This repository contains structured learning materials for the course  
**INT241: Data Structures and Algorithms**.  
The content is organized weekly, covering fundamental concepts, core data structures, and essential algorithms used in computer science.

---

## Course Objectives
- Understand fundamental data structures and algorithms
- Analyze time and space complexity
- Apply appropriate data structures to real-world problems
- Develop problem-solving and algorithmic thinking skills

---

## Course Outline

### Week 01: Introduction to Data Structures
- What is a Data Structure?
- Importance of Data Structures
- Types of Data Structures (Linear & Non-Linear)
- Abstract Data Types (ADT)
- Data Structures vs Algorithms

### Week 02: Basic Data Structures
- Arrays
- Linked Lists
  - Singly Linked List
  - Doubly Linked List
  - Circular Linked List
- Comparison between Arrays and Linked Lists

### Week 03: Basic Algorithm Concepts
- Definition of Algorithms
- Algorithm Design Principles
- Time Complexity (Big-O Notation)
- Space Complexity
- Best, Average, and Worst Case Analysis

### Week 04: List, Stack, and Queue
- List Implementations
- Stack (LIFO)
  - Operations and Applications
- Queue (FIFO)
  - Types of Queues
  - Applications

### Week 05: Heap and Priority Queue
- Heap Concepts
- Min Heap and Max Heap
- Heap Operations
- Priority Queue
- Practical Applications

### Week 06: Hash Table (Dictionary)
- Hashing Concepts
- Hash Functions
- Collision Handling Techniques
- Time Complexity Analysis
- Real-world Use Cases

### Week 07: Tree
- Tree Terminology
- Binary Tree
- Binary Search Tree (BST)
- Tree Traversal Methods
  - Inorder
  - Preorder
  - Postorder

### Week 08: Balanced Tree
- Problems with Unbalanced Trees
- AVL Tree
- Rotations and Balance Factor
- Red-Black Tree
- Comparison of Balanced Trees

### Week 09: Graph and Tree Concepts
- Graph Definition
- Graph vs Tree
- Types of Graphs
  - Directed / Undirected
  - Weighted / Unweighted
- Graph Representations
  - Adjacency Matrix
  - Adjacency List

### Week 10: Sorting Algorithms
- Importance of Sorting
- Basic Sorting Algorithms
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
- Advanced Sorting Algorithms
  - Merge Sort
  - Quick Sort
- Complexity Comparison

### Week 11: Greedy Algorithms
- Greedy Strategy
- Characteristics of Greedy Algorithms
- Classic Problems
  - Coin Change
  - Activity Selection
  - Huffman Coding
- Advantages and Limitations

### Week 12: Dynamic Programming
- Dynamic Programming Concepts
- Overlapping Subproblems
- Optimal Substructure
- Top-down vs Bottom-up Approaches
- Classic Problems
  - Fibonacci
  - Knapsack
  - Longest Common Subsequence

### Week 13: Graph Algorithms (Part 1)
- Graph Traversal
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Applications and Complexity

### Week 14: Graph Algorithms (Part 2)
- Shortest Path Algorithms
  - Dijkstra’s Algorithm
  - Bellman-Ford Algorithm
- Minimum Spanning Tree (MST)
  - Prim’s Algorithm
  - Kruskal’s Algorithm

### Week 15: Review & Project
- Comprehensive Review
- Problem-Solving Techniques
- Sample Exam Questions
- Mini Project / Final Project
